# 🌸 灵汐 (LingXi) —— 灵山胜境 AI 数字人导游系统

> **“山形酷似灵鹫，灵汐伴君同游。”** > 灵汐是一款基于 **Qwen3-0.6B** 与 **RAG (检索增强生成)** 技术打造的智慧景区导览系统。她不仅拥有玄奘法师西行取经的历史厚度，更能通过多模态交互、情感感知与实时语音，为游客提供沉浸式的千年佛国游览体验。

------

## 🌟 核心亮点

- **深度知识溯源**：内置灵山胜境完整知识库，涵盖从唐代“小灵山”渊源到现代“灵山梵宫”艺术精髓。
- **轻量化大模型**：基于 **Qwen3-Embedding-0.6B**，针对 4090 显卡深度优化，响应延迟极低。
- **情感驱动交互**：实时分析游客情绪，动态驱动数字人表情变化，实现“有温度”的对话。
- **文化沉浸视觉**：UI 采用新中式水墨风格，将千年古迹与未来科技完美融合。

------

## 🛠️ 技术架构

代码段

```
graph TD
    A[游客端/管理端] -->|REST API| B[FastAPI 后端]
    B --> C{AI 核心引擎}
    C --> D[Qwen3-0.6B: 智能对话/情感分析]
    C --> E[ChromaDB: 景区知识语义检索]
    C --> F[Edge-TTS: 多音色语音合成]
    B --> G[SQLite: 对话记录与配置持久化]
    E --> H[Embedding: Qwen3-0.6B-Embedding]
```

### AI 能力栈

| **能力**       | **技术实现**             | **备注**                                                     |
| -------------- | ------------------------ | ------------------------------------------------------------ |
| **大模型引擎** | Qwen3-0.6B               | 支持本地私有化部署，确保数据安全                             |
| **知识检索**   | ChromaDB + SDPA 优化     | 采用 **SDPA (Scaled Dot Product Attention)** 绕过底层环境冲突，实现高性能检索 |
| **语音合成**   | Edge-TTS                 | 模拟自然人声，支持多音色切换（如：温柔导游、专业讲解）       |
| **底层加速**   | PyTorch 2.11 + CUDA 13.0 | 适配 RTX 4090，显存占用优化                                  |

## 🚀 快速启动

### 1. 克隆仓库与环境准备

Bash

```
git clone https://github.com/your-username/LingXi-AI.git
cd LingXi-AI/backend
pip install -r requirements.txt
```

### 2. 配置模型与变量

将解压后的 **Qwen3-Embedding-0.6B** 放入 `models/` 目录：

Bash

```
7z x Qwen3-Embedding-0.6B.7z -o./models/Qwen3-Embedding-0.6B
```

编辑 `.env` 文件，确保路径正确。

### 3. 启动后端服务

Bash

```
# 建议通过 start.sh 或直接运行
uvicorn app.main:app --host 0.0.0.0 --port 5000 --reload
```

### 4. 启动前端页面

- **游客端**：`cd frontend-tourist && npm run dev`
- **管理端**：`cd frontend-admin && npm run dev`

## 🎯 核心功能清单

### 游客交互端 (LingXi Client)

- ✅ **文旅大模型对话**：基于 Qwen3 的深度景区知识问答。
- ✅ **多模态语音交互**：支持 Web Speech API 录音与实时语音播报。
- ✅ **灵汐数字人形象**：SVG 动态表情驱动，根据对话情感实时变换。
- ✅ **个性化路线推荐**：提供“历史文化深度游”与“自然风光全景游”等定制线路。
- ✅ **祈福互动**：模拟“抱佛脚”、“九龙灌浴”等互动仪式感。

### 管理后台 (LingXi Admin)

- ✅ **景区大脑大屏**：实时监控游客流量、服务量及满意度趋势。
- ✅ **游客情感画像**：分析游客反馈，生成情感分布饼图。
- ✅ **知识库 RAG 管理**：支持 `.md`/`.txt` 文档上传，实时更新导游知识。
- ✅ **AI 感受度报告**：一键生成游客游览感受深度分析报告。

## 🛡️ 环境配置与性能说明

- **推荐显卡**：NVIDIA RTX 4090 (24GB)。
- **注意**：为避免底层 C 库环境冲突，本项目默认使用 `attn_implementation="sdpa"`。
- **遥测说明**：为保护隐私，系统已默认禁用 ChromaDB 遥测：`Settings(anonymized_telemetry=False)`。
